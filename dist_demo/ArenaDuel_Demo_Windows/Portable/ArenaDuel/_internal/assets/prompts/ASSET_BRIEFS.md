# Arena Duel Visual Asset Briefs

These briefs define the target art direction for future external asset production.

## Global art direction

- fantasy arena
- readable in motion
- colorful but clean
- old-school anime MMORPG mood without copying any existing license
- cute undead mascots
- no gore
- no realism
- no photorealism

## 1. Main playable skeleton mascot

Use this prompt as a base for character creation:

"Create a 2D game sprite character for a top-down or semi top-down fantasy arena game. The character is a cute non-human skeleton mascot with a large skull, compact body, short limbs, readable silhouette, big eye sockets with magical glow, simple rib cage shapes, tiny boots and soft fantasy accessories. Style is colorful fantasy cartoon, clean, readable, polished indie game art, not realistic, not dark, not gory. The design must stay readable in combat from a distance. Provide idle and walk frames on transparent background. Team variants should work with warm ember accents and cool cyan accents."

## 2. Portrait of the skeleton mascot

"Create a bust portrait of the same cute fantasy skeleton mascot used in a 2D arena game. Large skull, friendly expression, stylized magical eyes, fantasy accessory like a tiny circlet, hood, scarf or charm. Clean cartoon fantasy style, readable at small size, transparent background, polished UI portrait asset, non realistic, non violent."

## 3. Arena icon / app icon

"Create a game icon for a fantasy arena game called Arena Duel. The icon should feature the cute skeleton mascot head as the main subject. Large skull silhouette, strong outline, bright magical eyes, simple fantasy accessory, readable at very small size, premium game icon style, centered composition, clean contrast, no text, no realism, no gore, transparent or icon-friendly background."

## 4. Forgotten Sanctum arena floor

"Create a top-down fantasy arena floor for a stylized 2D game. Theme is forgotten sanctum with light undead flavor, ancient runes, worn stone, soft magical glow, ruined shrine atmosphere, readable lanes for competitive play, clean navigation shapes, not cluttered, not realistic, not grimdark. Use a polished cartoon fantasy style suitable for a readable action game."

## 5. Arena obstacles set

"Create a small top-down fantasy obstacle set for a stylized 2D arena game. Include ruined walls, round stone pillars, grave markers, rune stones and shrine fragments. Shapes must be readable from above, collisions should feel obvious, silhouettes clean and gameplay-friendly, colorful but grounded fantasy palette, non realistic, no gore."

## 6. UI ornaments

"Create fantasy UI decoration pieces for a polished arena game interface. Include banner trims, filigree corners, soft magical dividers, rune medallions and subtle panel overlays. Style should feel premium, readable, elegant, fantasy game UI, not cluttered, not photorealistic."

## Replacement rules

- keep transparent backgrounds for sprites, portraits and UI pieces when possible
- keep filenames stable once integrated into code
- keep asset sizes in powers or multiples that scale cleanly in pygame
- prefer readable silhouettes over detail noise
